# FA2 Multi Nft Asset Multi Admin Contract deployment using taquito


![Built with Taquito][logo]

A minimal React setup for starting developing Tezos Contact quickly with Taquito by connecting tezos wallet.

## Getting Started

1. Make sure you have https://nodejs.org/ installed on your computer
2. Create a new repository from taquito-boilerplate by clicking "Use this template".
3. Clone your new repository:

   `git clone <YOUR_REPOSITORY_URL>`

3. Change your current working directory to the newly cloned repository directory.
4. Install dependencies:

   `npm install`

5. Start development server:

   `npm run start`

6. Open https://localhost:3000 in your browser to see a sample application.
